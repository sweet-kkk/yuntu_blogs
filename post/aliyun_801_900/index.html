801. 黑客对网站进行扫描，发现页面存在注入风险，通过手工构造SQL注入语句，渗透入最客户的效据库,获取网站相关核心数据。可以通过（）来进行解决

选项:

  • A. 安全管家
  • B. web应用防火墙
  • C. 使用正则表达式过滤传入的参数
  • D. 云安全中心

正确答案: BC

解析:

答案是B和C。针对 SQL 注入问题，可以通过 Web 应用防火墙 (WAF) 进行解决。WAF 可以提供基于规则的 SQL 注入攻击防御，通过对用户请求进行监控和过滤来防止 SQL 注入攻击。正则表达式过滤传入的参数也可以为应用程序提供基本的防护机制，通过对请求参数进行过滤，检查是否含有特殊字符等。安全管家是一个安全扫描工具，可以扫描和修复云资源中的安全漏洞，但无法解决 SQL 注入问题。云安全中心是一个多云安全管理平台，可以进行企业级安全管理和威胁检测等，但也无法解决 SQL 注入问题。因此，选项B、C是正确的描述。

802. 在RDS只读实例变更配置过程中,RDS主实例无法进行哪些任务

选项:

  • A. 查看SQL审计日志
  • B. 内外网切换
  • C. 备份实例
  • D. 创建实例

正确答案: BCD

解析:

B. 内外网切换：由于只读实例变更配置期间的复制延迟，RDS 主实例无法在此期间执行内外网的切换操作。C. 备份实例：由于备份实例的创建过程可能需要对数据库进行重启或复制，因此在只读实例变更配置期间会被禁用。D. 创建实例：在 RDS 只读实例变更配置过程中，RDS 主实例可能会被锁定，无法执行新实例的创建操作。因此，答案为 B、C、D。A 选项中的查看 SQL 审计日志不受影响，可以在变更过程中执行。
